# Project Context

- **Owner:** Ricky Kaare Engelharth
- **Project:** Cabazure.Client — a .NET library for generating strongly-typed HTTP clients via C# Source Generators
- **Stack:** C# / .NET 9, NuGet package, Azure.Core / Azure.Identity, ASP.NET Core DI integration
- **Created:** 2026-03-10

## Learnings

- Primary docs: `README.md` at repo root
- Extended docs: `docs/` directory
- Sample project: `samples/AzureRest/` — the reference implementation for how consumers use the library
- Public API surface lives in `src/Cabazure.Client.Runtime/` — attributes, response types, options interfaces
- Key consumer-facing types: `[ClientEndpoint]`, `[Get]`/`[Post]`/`[Put]`/`[Delete]`, `[Path]`/`[Query]`/`[Header]`/`[Body]`, `EndpointResponse<T>`, `PagedResponse<T[]>`, `ClientRequestOptions`
- Generated DI extension: `AddCabazureClient()` — generated by `ClientInitializationGenerator`
- Authentication: `AzureAuthenticationHandler` wraps `DefaultAzureCredential` via `Azure.Identity`
